```rust
fn main(){
    println!("Hello, world!");
}
```

```terminal
$ rustc main.rs
$ ./main
Hello, world!
```
println! calls a Rust macro
a ! means that you’re calling a macro instead of a normal function and thatmacros don’t always follow the same rules as functions
